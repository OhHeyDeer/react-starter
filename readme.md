# Nicks Movie List

## Basic CRUD Application
* Front End Application Only -- This application is designed as a basic CRUD learning app.
* I learned to Create, Read, Update, and Delete components or information from my page.
* The app is a basic movie list which can store movie names and list them out on the page. 
* Does not persist data in any was as the application is purely a frontend experiment.
